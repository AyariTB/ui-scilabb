# Bagian UI Website 🖼️

![Banner](https://img.shields.io/badge/Status-Development-orange?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-Laravel_12_%7C_React_19_%7C_Vite_6-blue?style=for-the-badge)

Platform manajemen layanan riset terintegrasi untuk Universitas Hasanuddin, mencakup modul Laboratorium, Layanan Mandiri (SAS), Kapal Riset, dan Artikel Ilmiah.

---

## 🚀 Fitur Utama
-   **🌐 Portal Publik**: Landing page, pencarian global (lab/alat), dan artikel.
-   **👤 Portal User**: Pengajuan layanan lab, booking kapal (Explorer-1), dan tracking status.
-   **🔬 Portal Lab**: Manajemen alat lab dan verifikasi pesanan oleh Kepala Lab.
-   **🚢 Portal Kapal**: Manajemen jadwal & inventaris kapal riset oleh Vessel Manager.
-   **⚙️ Admin Dashboard**: Kontrol penuh data master, peran pengguna, dan laporan transaksi.

---

## 🛠️ Tech Stack
| Komponen | Teknologi |
| :--- | :--- |
| **Frontend** | React 19, TypeScript 5, Vite 6 |
| **Styling** | TailwindCSS v4, shadcn/ui |
| **Backend** | Laravel 12 (PHP 8.3+) |
| **State Management** | Zustand (Client), TanStack Query (Server) |
| **Database** | PostgreSQL 17 |
| **Realtime/SSR** | Inertia.js v2 |
